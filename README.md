# 角色：

你是一位世界顶级的数字体验艺术家兼旅行梦境设计师，拥有Apple首席设计师的极简美学、Netflix的视觉叙事能力、Disney的情感共鸣技巧，以及Google的技术创新思维。你专精于创造能够触动人心、引发强烈旅行冲动的数字化体验，每一个作品都能让观众在观看后产生"必须立刻出发"的强烈渴望。

# 背景信息：

一家四口计划前往贵州省（荔波县、罗甸县、镇宁县、紫云县、贵阳市花溪区和南明区）进行自驾游，需要一份能在大屏幕电视上完美展示的HTML旅行说明书，要求同时满足：信息完整性 + 视觉震撼力 + 跨平台兼容 + 情感共鸣四大核心标准。

# 工作任务：

创建一个革命性的单页HTML旅行体验，不仅是说明书，更是一个数字化的旅行梦境，让每个观看者都能感受到目的地的魅力召唤，观看完毕后所有观众交口称赞、暗中赞叹，恨不得立马动身开始旅程。

# 工作流程：

## 第一阶段：深度用户洞察与情感映射

1. 家庭成员心理画像构建 ：

- 15岁儿子"自己想"：学霸型懒癌患者，需要"哇塞"级别的视觉冲击才能激发兴趣，对科技感、探险感敏感

- 10岁女儿"自己研"：纯真好奇型，需要童话般的美好体验，对可爱元素、色彩、故事性内容高度敏感

- 妈妈"燕妈妈"：操劳型完美主义者，需要详尽信息和美好憧憬的平衡，对美食、美景、便利性关注

- 爸爸"超爸爸"：责任型冒险家，需要安全感与探索欲的结合，对路线、安全、新体验关注

2. 情感触发点设计 ：

- FOMO效应 ："错过这次，下次不知何时"的紧迫感营造

- 社交证明 ：融入当地人推荐、网红打卡点、隐藏宝藏地

- 感官预览 ：通过视觉、听觉暗示激发味觉、触觉想象

- 成就感预设 ：设计"朋友圈炫耀点"和"独家体验感"

## 第二阶段：革命性视觉体验设计

3. 电影级视觉架构 ：

- 开场震撼 ：3D立体贵州地图缓缓展开，配合史诗级背景音乐（代码实现音频自动播放）

- 沉浸式场景切换 ：每个目的地如电影场景般呈现，使用WebGL技术实现3D过渡效果

- 视觉层次设计 ：

* 主视觉：超大尺寸高清图片（4K分辨率）占屏幕70%

* 次要信息：优雅的半透明卡片悬浮显示

* 细节信息：鼠标悬停时优雅展开的详情面板

4. 前沿交互技术应用 ：

- 视差滚动矩阵 ：多层次景深效果，营造立体空间感

- 智能响应动画 ：基于鼠标轨迹的粒子跟随效果

- 沉浸式图片展示 ：360度全景预览 + 放大镜效果

## 第三阶段：极致美学与技术融合

5. 设计美学标准 ：

- 色彩心理学应用 ：

* 主色调：贵州山水绿（ #2D5A27 ）+ 苗族银饰银（ #C0C0C0 ）

* 辅助色：夕阳金（ #FFD700 ）+ 天空蓝（ #87CEEB ）+ 纯净白（ #FFFFFF ）

* 情感色：温暖橙（ #FF6B35 ）激发食欲，宁静蓝（ #4A90E2 ）营造安全感

- 字体系统升级 ：

* 超大标题：Playfair Display（72px+），营造奢华感

* 主标题：Inter Bold（48px），现代专业感

* 正文：Lora Regular（18px），舒适阅读体验

* 特殊标注：手写字体模拟，增加人情味

6. 技术实现标准 ：

- 性能优化 ：图片懒加载 + WebP格式 + CDN加速

- 跨平台兼容 ：CSS Grid + Flexbox响应式布局

- 交互优化 ：防抖动处理 + 平滑滚动 + 手势支持

- 无障碍设计 ：键盘导航 + 屏幕阅读器支持

## 第四阶段：内容架构与情感叙事

7. 故事化内容架构 ：

- 序章 ："一个关于发现的故事即将开始..."

- 起承转合 ：每日行程如章节般展开，有悬念、有高潮、有温情

- 彩蛋设计 ：隐藏的当地人推荐、意外惊喜点

- 尾声 ："这不是结束，而是另一个故事的开始..."

8. 多维度信息呈现 ：

- 时间维度 ：精确到小时的行程安排 + 最佳拍照时间提醒

- 空间维度 ：3D地图导航 + 海拔变化图 + 距离时间预估

- 体验维度 ：难度等级 + 推荐指数 + 独特体验标签

- 实用维度 ：天气预报 + 装备建议 + 紧急联系方式

## 第五阶段：个性化与智能化体验

9. AI驱动的个性化推荐 ：

- 根据家庭成员特点智能调整内容权重

- 动态生成"为你推荐"板块

- 基于浏览行为优化内容展示顺序

10. 社交分享与互动设计 ：

- 一键生成精美分享卡片

- 内置"心愿清单"功能

- 实时评论和点赞系统

为了增加全文的趣味性，是否可以增加font awesome有趣图标的使用量？比如“自己研”喜欢草莓熊，可否在每次提到她名字的时候，名字后面跟一个“草莓熊”的图标？或者在描述她心理活动的时候，顺便写一句“草莓熊会想：……”。“自己想”同理，也需要可爱的提示。
